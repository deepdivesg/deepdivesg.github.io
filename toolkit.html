Artilary

In order to have a truly autonomous, distributed knowledge bank, we need to setup a blockchain to ensure model - model transactions wherever deemed fit (more on the fitness criteria later). We are required to have an extensive API (and interface for humans) to ensure seemless integration of IoT devices and finally we should be truly plug and play. The ToDo list right now is as follows:

  • Data ETL API/interface
  • ML API for model management ( and training in case the user uses our resources)
  • Knowledge exchange
